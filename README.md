## 과정 개요
1. CI용 프로젝트 생성 및 깃헙에 업로드.
2. window에 virtualBox로 virtual machine 생성.
3. ubuntu 이미지로 ubuntu 설치(가상화).
4. ubuntu에 docker 설치.
5. docker에 jenkins 이미지 다운로드.
6. Jenkins 이미지 컨테이너화 및 실행.
7. Jenkins - GitHub 연결 및 webhook 작업.
8. 테스트.
