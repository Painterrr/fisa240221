## 과정 개요
CI용 프로젝트 생성 및 깃헙에 업로드.
</br> **↓** </br>
window에 virtualBox로 virtual machine 생성.
</br> **↓** </br>
ubuntu 이미지로 ubuntu 설치(가상화).
</br> **↓** </br>
ubuntu에 docker 설치.
</br> **↓** </br>
docker에 jenkins 이미지 다운로드.
</br> **↓** </br>
Jenkins 이미지 컨테이너화 및 실행.
</br> **↓** </br>
Jenkins - GitHub 연결 및 webhook 작업.
</br> **↓** </br>
테스트.
